Computers, the Internet, and HTML

HyperText Markup Language

Hypertext Markup Language, or HTML, is the basic language structure that a computer uses to comunicate with the internet and other computers in order to display web pages, images, videos, games, anything. It instructs what and how text is displayed, where pictures and videos live, and so on.

The Internet

When you, say, click on a link, the computer sees the link as HTTP (HyperText Transfer Protocol), which it sends to another computer, known as a server. When the server finds the appropriate webpage, image, etc., it sends your computer HTML to instruct it on how to display the information.

HTML Structure

Each individual piece within HTML is called an element. Each element has a tag, which denotes what it is and what it does. Tags can be used to make the text bold, italicized, underlined, struckthrough create links, and various other things. An element takes the form: < tag >content< /tag >. The / denotes that it is an end-tag - without it, the tag would run straight to the end. So, instead of just one word, you'd make half your sentence bold.

You Are Smarter Than the Computer

The old saying says computers are only as smart as the people who program them. This is 100% correct. Computers are purely proceedural; they cannot interpret what they see as anything other than what it is. If you forget a bracket or mistype a URL (web address), your page will not look the way you wanted.

Two Types of Elements

HTML elements can be either inline or block. Block elements place the content within an invisible box, which separates it from the rest of the content on the page. In general, inline elements are used to modify the look of the text, while block elements modify how the text appears in the page.
